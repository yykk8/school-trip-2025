# school-trip-2025
Trip Notes ― 自分たちで作る世界一の修学旅行 ―修学旅行の思い出や学びを記録・共有するためのポートフォリオサイトです。「楽しいだけで終わらせない」をコンセプトに、生徒たちが撮影した写真や学習成果をクラス横断でアーカイブします。📸 機能一覧ヒーローセクション: 東京と山梨の風景が切り替わるスライドショー学びの共有 (Archive): クラス別の学習成果フィルタリング機能ギャラリー: 「風景」「体験」「食事」などのカテゴリ別写真閲覧旅の軌跡: 次年度の生徒へ向けた引継ぎ・ガイド機能リアクション: 投稿への「いいね」機能（ローカル動作）🚀 使い方このリポジトリをダウンロード（Clone）します。index.html をブラウザで開くだけで動作します。GitHub Pages を設定の設定で Source を main ブランチ（または /root）にすると、Webサイトとして公開できます。🛠 使用技術HTML5 / CSS3 / JavaScript (Vanilla JS)Tailwind CSS (CDNで読み込み)Google Fonts (Zen Maru Gothic, Noto Sans JP)Unsplash (サンプル画像)📁 ファイル構成このプロジェクトは、管理を容易にするため「シングルファイル構成（HTML内にCSS/JSを内包）」を採用しています。.
